Introduction

Almost every person in the world knows what the Internet is and most know how to use it, but only a few know how it works. This is documentation that will cover just one aspect of the components that make up the Internet.

This section is called front end web development and consists of HTML, CSS and goes hand in in with Javascript which are the three parts to most front end web development.

HTML and CSS are the back bone of the the world wide web as we know it. HTML gives a web page structure and the CSS gives it the visual appeal.

Both work together to create the webpages that we use everyday. They are completely different from each other. One is reliant on the the other, but the other compliments. We will get into those differences.

  • HTML (Hypertext Markup Language) is a markup language that allows website creation through hyperlinks that you can click to go to any place on the web. It is used to design the text that it will distinguish as the different types of text in a webpage.
  • CSS (Cascading StyleSheet) is a language for making the web document more asthetically pleasing with CSS elements that alter the look of the HTML in use on the webpage.
HTML Basics

The core tools of HTML are the elements (< > text < / >), which are the symbols that enclose a command that is called a tag. These tags tell the text how to behave and the content that goes in between them i.e. "text" make of an element.

  • Opening Tag and Closing Tag are "< p >" and "< /p >", which tells the document what the content that goes in between this tag will be or do. In this example it will be a paragraph. The slash is required to correctly close a tag.
  • Content is basically what goes in between the tags and in the example it is the word "text".
CSS

In CSS the main elements in use are CSS selectors which are " { }". This is implying that the HTML element that we want to style is going be described inside the curly braces, where the alterations take place such as color or font style.

  • Selectors are the way CSS pick which elements are going to be changed. This is saying that the pareagrah element will be in blue.
    p { color: blue }
  • The declaration consists of the rule that is being changed. In this case is the part that says " color: blue ".
  • The property is the name for the item that we want to change. i.e. color is a property. The actual color itself "blue" is the property value.
Syntax

Syntax is the way that a language is supposed to be written such as the rules and the order. That's no different in HTML and CSS. The examples for both HTML and CSS are the syntax involved in each language.


This is some syntax in HTML:
 <h1>Hello World</h1>
This is some syntax in CSS:
 <h1>{background-color: blue;}</h1>

The CSS Exception

The one big difference in the use and application of CSS is the fact that CSS can be added to an html file directly. It can also be written in a seperate file and that file can be reference in the HTML file, however, html cannot be written in a CSS file.

This exception is referenced to as inline style. This is when you are adding the CSS style inside the HTML file.

 <head>
 <title>My HTML Document</title>
 <style>
body { background-color: YellowGreen; }
p { color: #fff; }
</head>
</style>
